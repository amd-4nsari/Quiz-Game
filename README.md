🎯 Quiz War – Multiplayer Quiz Game

Quiz War is a browser-based multiplayer quiz game built using HTML, CSS, and JavaScript.
It allows two players to compete in real-time quiz battles while teachers can manage custom questions through a secure dashboard.

The project is fully frontend-based and uses localStorage for persistent data storage, meaning questions, player stats, and settings remain saved even after the browser tab is closed.

✨ Features
⚔️ 2-Player Quiz Battle System
🧑‍🏫 Teacher Dashboard
🔐 Password Protected Admin Panel
➕ Add / Delete Custom Questions
📚 Multiple Subjects & Difficulty Levels
📊 Player Statistics & Accuracy Tracking
🏆 Win Tracking System
🔊 Sound Effects & Interactive UI
💾 Data Persistence using localStorage
📱 Responsive Design
🎨 Modern Game-Style Interface
